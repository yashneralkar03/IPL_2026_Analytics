# IPL 2026 Batter Performance Analysis

## Overview

Ball-by-ball analysis of IPL 2026 batting performance using Python and Power BI.

## Key Analyses

- Phase-wise strike rates (Powerplay, Middle, Death overs)
- Pace vs Spin performance splits
- Top performers by phase in IPL 2026

## Tools Used

- Python (pandas)
- Power BI
- Data Source: Cricsheet.org

## Files

- `notebooks/analysis.py` — data cleaning and analysis script
- `output/` — processed CSVs used in Power BI dashboard
